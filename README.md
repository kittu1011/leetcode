# LeetCode

Personal solutions to LeetCode problems, organized by topic.

## Structure

| Folder | Topic |
|--------|-------|
| `arrays/` | Array manipulation, prefix sums, sorting |
| `binary_search/` | Binary search and variants |
| `dynamic_programming/` | Memoization, tabulation, classic DP patterns |
| `graphs/` | BFS, DFS, topological sort, union-find |
| `linked_lists/` | Singly/doubly linked list problems |
| `sliding_window/` | Fixed and variable window problems |
| `trees/` | Binary trees, BSTs, tries |
| `two_pointers/` | Two pointer and fast/slow pointer problems |

## File naming

```
{problem_number}_{problem_name}.py
```

Example: `0042_trapping_rain_water.py`
